## TROUBLESHOOTING

### "Module not found" errors:
- Lovable auto-installs dependencies
- Refresh editor if errors persist

### "Component not found" errors:
- Verify shadcn/ui components exist
- Lovable should have Button, Card, Input by default

### Routing not working:
- Check `react-router-dom` in dependencies
- Verify BrowserRouter wrapping in App.tsx

### Blank page:
- Check browser console for errors
- Verify all imports are correct
- Try hard refresh (Cmd+Shift+R)
