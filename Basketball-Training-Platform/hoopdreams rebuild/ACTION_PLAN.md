## üî• IF SOMETHING BREAKS

**Don't panic!** Here's what to do:

1. **Check browser console** - Red errors tell you what's wrong
2. **Verify imports** - Make sure all paths start with `@/`
3. **Confirm dependencies** - Lovable should auto-install
4. **Try hard refresh** - Cmd+Shift+R clears cache
5. **Check Lovable build logs** - Bottom of editor shows errors

**Common fixes:**
- Missing imports? Add them at top
- Component not found? Check shadcn/ui
- Routing broken? Verify react-router-dom
- Styles not working? Check Tailwind classes
