# BLAST Curriculum - Master Drill Index

Quick reference for all drills across all levels.

---

## By Level

### Travel (Youth: 2nd-8th Grade)
Foundation drills focusing on fundamentals, rhythm, and basic movement patterns.

| Category | Key Drills |
|----------|------------|
| Spin Cycle | Triple Threat Pivots, Pound Pivot Cross |
| Finish | Basic Layups, Scoops, 2-Ball Scoop |
| Handles | Pound, Cross, Between, Behind, Front Cross, L Dribble, Inside Out |
| Handles (Move) | Speed Dribble, Front Cross, Retreat Dribble |
| The Box | Retreat Dribble (12 count) |
| Combos | Pound Between Back Through, Pound Between Behind Cross |
| Tennis Ball | Pound Behind, 2-Ball Inside Out |
| Tennis + BB | Cross Underneath Grab |
| Passing | Chest, Bounce, Backdoor, Post Feed, Lead Pass |
| Pivot/Jab | Forward/Reverse Pivots, Jab Steps |
| Step Backs | Step Back On Dribble |
| Side Step | Basic Side Step Jump Shot |
| Shooting | Form Shooting (3 swishes), Flare Off Pin Down |
| Footwork | Stutter, Big Step Little Step, Punch Drag, Load Explode |
| Ball Screen | Under, Hip Pocket, Hard Hedge Reads |

---

### Varsity (High School: 9th-12th Grade)
Intermediate drills adding complexity, 2-ball work, and combination patterns.

| Category | Key Drills |
|----------|------------|
| Spin Cycle | Pound Hop, Spin, Punch |
| Finish | 2-Ball Scoop, Inside Hand, Pro Hop, Euro Step, Reverse |
| Handles | 2-Ball drills, L Dribble patterns, Catch/Pass combos |
| Handles (Move) | Inside Out (motion), Punch Drag |
| The Box | Retreat Dribble (12 count) |
| Combos | Pound Between Behind (3 count) |
| Tennis Ball | Pound Between patterns, Inside Out Between Cross |
| Tennis + BB | Between Grab, Behind Grab |
| Passing | One-handed passes, Off-dribble, Thread Needle, Wing-to-cutter |
| Pivot/Jab | All pivots and jabs |
| Step Backs | Step Back On Dribble |
| Side Step | Basic Side Step Jump Shot |
| Lock Up | 45-sec Isometric, Defensive Shuffle patterns |
| Shooting | Form Shooting (6 swishes), Flare Off Pin Down |
| Footwork | Stutter, Big Step Little Step, Punch Drag, Load Explode |
| Ball Screen | Under, Hip Pocket, Hard Hedge Reads |

---

### Recruit (College-Bound: 9th-12th Grade Advanced)
Advanced drills with complex sequences, elite coordination, and game-situation reads.

| Category | Key Drills |
|----------|------------|
| Spin Cycle | Behind Back Spin, Between Legs Spin, Half Spin Shimmy |
| Finish | Euro Step, Outside Foot Plant |
| Handles | Advanced patterns (shared with Varsity) |
| Handles (Move) | Wrap, Drop Cross, Underdrag |
| The Box | Retreat Dribble, LH/RF Step Cross variation |
| Combos | Cross Cross Through, Between Back Through Cross, 9-dribble sequence |
| Tennis Ball | Inside Out Between Cross |
| Tennis + BB | Pound Between/Behind Grab Cross |
| Passing | Off-hand passes, Back passes, Pocket Pass Ball Screen |
| Pivot/Jab | All pivots and jabs |
| Step Backs | Step Back On Gather, Step Back Cross Step Back |
| Side Step | All side step combinations |
| Lock Up | 90-sec Isometric, Defensive Shuffle patterns |
| Shooting | Form Shooting (9 swishes), Curl Off Pin Down (all variations) |
| Footwork | Stutter, Big Step Little Step, Punch Drag, Load Explode |
| Ball Screen | Under, Hip Pocket, Hard Hedge Reads |

## Quick Practice Builder

**10-Minute Warm-Up (Any Level):**
1. Stutter (full court) - 2 min
2. Pound dribbles (waist high, shoulder high) - 2 min
3. Cross alternating - 2 min
4. Between/behind patterns - 2 min
5. Form shooting (T3/V6/R9) - 2 min

**30-Minute Skill Session:**
1. Footwork patterns - 5 min
2. Handle drills (stationary) - 5 min
3. Handle drills (moving) - 5 min
4. Finishing drills - 5 min
5. Shooting drills - 5 min
6. Ball screen reads - 5 min

**Focus Day Options:**
- **Ball Handling Day:** Categories 3, 4, 5, 6
- **Finishing Day:** Categories 1, 2, 15
- **Shooting Day:** Categories 11, 12, 14
- **Passing Day:** Category 9
- **Defense Day:** Category 13
- **Complete Game:** Categories 14, 15, 16
