## Tool Philosophy

**Built when needed:**
- Real problems with real consequences
- Faster to build than to find/adapt
- Used immediately in production
- Improved based on real feedback

**Design principles:**
- Single-file when possible (easy to share)
- Works offline (no internet required)
- Coach-friendly interface (no technical knowledge needed)
- Printable output (for gym bags)

**Not trying to:**
- Replace commercial software
- Build a business
- Create the perfect solution
- Solve every problem

**Just trying to:**
- Make game day less stressful
- Save time on routine tasks
- Avoid preventable mistakes
- Help coaches focus on coaching

---

## Tool Standards

**Each tool should have:**
- Clear README with setup instructions
- Actual usage examples
- Troubleshooting section
- Potential improvements list
- Version number and date

**Good tools are:**
- Simple enough to use immediately
- Documented well enough to share
- Tested in real situations
- Improved based on actual use
