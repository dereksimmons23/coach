## Key Strategic Themes

### From Kabir's App to HoopDreams Platform

**The origin story:**
- Built simple training app for Kabir's India trip
- Realized it solves solo training challenge globally
- Discovered potential to scale coaching methodology
- Vision evolved from MVP to platform concept

**Platform possibilities:**
- "ReciMe for basketball drills"
- Clip drills from social media
- Organize personal drill library
- Build practice plans
- AI coaching assistance (CoachFlo)

**Current decision:** Repository first, platform later (maybe)

### Neurodivergent Athlete Coaching

**Market gap identified:**
- 10%+ of youth athletes are neurodivergent
- No existing platforms address this specifically
- Coach D has specialized expertise
- Opportunity for differentiation and mission alignment

**Research questions:**
- How does ADHD/autism affect skill acquisition?
- What coaching adaptations work best?
- How to scale specialized coaching knowledge?

**Connections to broader work:**
